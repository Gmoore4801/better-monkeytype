How does Better MonkeyType work?

Everything is automated by Better MonkeyType to make it straighforward for the user. There are 4 phases spread out over 15 minutes of typing time:

  1. Testing (0-5): Your accuracy from this phase determines the letter that you practice in the next phase. You can check your stats in the console at the end of this phase. The language used is english 5k.
  2. Training (5-10): The letter that you missed the most during testing will be in every word in this phase for optimal practice. The language used is english 5k.
  3. Practice 1 (10-13): This phase practices a smaller set of more common words to help develop muscle memory. The language used is english 1k.
  4. Practice 2 (13-15): This phase uses a set of the 200 most commonly used words. You will eventually develop muscle memory for these words and type them faster, allowing you to practice typing at a faster speed.

FAQs

How to access the console: Hold ctrl (cmd for mac) + shift + i, then click "console" on the top navigation bar.

What are the different languages in MonkeyType? MonkeyType has different word groups that are larger than the default for the language. The default includes 200 words which is not enough to practice for real world uses, so Better MonkeyType uses larger word groups. You can change these manually by pressing escape and typing "language".
